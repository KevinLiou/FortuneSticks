雷雨師一百籤


第十六籤【下下。乙己】

官事悠悠難辨明
不如息了且歸耕
傍人煽惑君休信
此事當謀親弟兄
（傍字，另有作旁字一說）




聖意一

訟難明。和為貴。名利無。行人至。婚未成。病有祟。財莫貪。終無利。


聖意二

訟難明。和為貴。功名無。行人至。婚未成。病有祟。財莫貪。終無利。


東坡解

訟事未決。且宜從和。傍人煽惑。平地風波。

慎勿輕信。自投網羅。謀之兄弟。誰能奈何。


碧仙註一

埋頭莫向前。時違休怨天。

貴人垂手援。亨通到百年。


碧仙註二

埋頭莫向前。時違休怨天。貴人垂手接。亨通到百年。


解曰

此籤訟事未辨。以和為貴。名利全無。婚姻未當。

凡有謀望。且待時來。有人煽惑。勿輕信也。


釋義

問訟顯然宜和。士人占之。功名未遂。

官員占之。主難進步。有礙前程。

蓋明有神非。幽有鬼責。財利難求。營謀寡效。

凡事待時。不宜聽信讒人。萋菲之說。

兄弟謂至親之人。當謀之以決疑惑也。


占驗

一監生科舉。卜此。見遺。後其弟新授。例到監。偕弟同懇人司成。

得以續取。此事當謀。句驗甚。


故事

王祥臥冰（與本籤較無關聯，驗證者仍可參考）

晉。王祥字伏徵。臨沂人。性至孝。事繼母極恭。

謹母欲食生魚。天寒冰凍。祥赤身臥於冰上。求之。

其冰立解。得雙鯉。持歸。人以為孝感云。

後官至太保。進公爵。

(廿四孝)
