雷雨師一百籤


第十一籤【下下。乙甲】

今年好事一番新
富貴榮華萃汝身
誰道機關難料處
到頭獨立轉傷神
（道字，另有改識字一說）




聖意

莫貪求。名未遂。財祿平。訟不利。病者凶。事不濟。問行人。多阻滯。


東坡解

命運難亨。亦不如意。吉中有凶。事實無濟。切莫貪求。出外不利。如或妄動。福消禍至。


碧仙註一

作事須宜慎。人心隔肚皮。用心防算險。百事自合宜。


碧仙註二

作事須宜慎。人心隔肚皮。用心防算險。百事允相宜。


解曰

此籤宜更改方得財祿。要知足。

不可與人其事。恐機事不密。反成禍敗。

官有旌麾之喜。孕生男。士有超拔之喜。訟者勝。婚不吉。


釋義

此籤卻要知道吾人處世。當見機而作。今年命運。雖吉未吉。似亨未亨。

欲問孕生男。終難長養。底於有成。難料處言。事有變幻。

凡事占之徒勞心力。若強為。到底無益。立字一作力字。


占驗

一生到省。考遺才。占此。不錄。至返家時。乏騎步行。應轉傷神也。

又兩商人同往茶山採茶。多年蝕本。欲收場。未決。故托人占此。

或勤再做。一信之。一不聽。遂折股。

其一人將資本攜歸。中途得病。到歲杪。尚在旅寓獨宿。末句驗甚。

一人再整適茶葉。失收。但人心太寒。無敢買。被獨廣購。果大有獲。

改因轉賣(賣本作左貝右僕心僕去人)洋船。乃敗。

全詩俱應。末句到頭獨立。功極。轉傷神三字尤妙。



故事

(一)韓信功勞不久

漢韓信。准陰人。佐漢高帝定天下。虜魏。破趙。降燕。下齊。滅楚。建十大功勳。初封楚王。高祖深忌其能。後偽遊雲夢。執歸降。封淮陰侯。乃不自損抑。務伐己功。致有殺身赤族之禍。亡年僅三十二歲。雖有功勞。不久於世。

(二)孫臏遇龐涓

戰國。孫臏齊人。龐涓魏人。同師事鬼谷子。孫臏之學優於龐涓。涓為魏將。與臏鬥智。輸而忌之。乃以計刖其足。後孫臏假作顛狂。得脫歸。齊威王以(臏)為師。將兵伐魏。用減灶添兵之法。賺龐涓。追至馬陵道。伏弩射死。
