雷雨師一百籤


第三四籤【中平(中吉)。丁丁】

春夏纔過秋又冬
紛紛謀慮攪心胸
貴人垂手來相援
休把私心情意濃
(援字，另有作接字一說)




聖意一

訟有憂。病未瘳。財祿散。忌遠謀。

行人動。婚莫求。防口舌。免悶愁。


聖意二

訟解憂。病漸瘳。財祿有。忌遠謀。

行人動。婚莫求。防口舌。免悶愁。


東坡解一

一歲營謀。未有發達。必得貴人。垂手提挈。

漸漸亨通。可免激聒。莫信私人。巧語鬥謀。


東坡解二

一歲營謀。未有發達。方得貴人。垂手提挈。漸漸亨通。可免激聒。莫信私人。巧語鬥諜。


碧仙註一

訟則憂兮病不瘳。財物耗散莫相謀。

行人動作婚難就。口舌須防免唧啾。


碧仙註二

訟解憂兮病漸瘳。財物凋零莫妄求。

行人動作婚難就。口舌須防免唧啾。


解曰

此籤空虛。指望無成。徒勞自己心力。

謀事不就。所為虛費功夫。營求未有發達。

若得貴人提攜。漸漸亨通。

莫信私言巧語。方免聒聒也。


釋義

一年之內。謀為紛紛。徙然勞苦。若得貴人提挈。

只宜迸去私慮。將心靠他。始得亨通。

若惑于婦人之言。或因他事之累。則彼此之間。互相猜忌。

事便無益。占者得之。防口舌。名利休問。


占驗

一人出外抽豐。其父在家。占此。

其手子客居數年。竟空手而回。所得盡付花柳(煙花柳巷)之費矣。



故事

蕭何追韓信

漢。蕭何。沛人。

韓信。淮陰人。

時漢王燒絕棧道。部下將士逃亡甚多。其時。信歸漢逃去。

何知之。連夜追信回。翌日王知。怒曰將士逃亡過半。

不聞丞相有所追。獨追信。何也。

何曰大王如無意東伐。固罷。

若與楚爭天下。非信莫當其任。乃拜信為大將。
