雷雨師一百籤


第九八籤【中平(中吉)。癸辛】

經營百出費精神
南北奔馳運未新
玉兔交時當得意
恰如枯木再逢春




聖意

名利有。晚方成。訟與病。久方平。

孕生子。行阻程。遇卯運。事皆亨。


東坡解一

營謀難力。況逢時否。奔北趨南。百事未遂。

卯年月日。方可營計。轉瘁為榮。節節生意。


東坡解二

營謀艱力。況逢時否。奔北趨南。百事未遂。卯年月日。方為營計。轉悴為榮。節節生意。


碧仙註一

買賣弗精神。求謀時運平。

卯年月日至。方得稱心情。


碧仙註二

前路蹉跎幾歎嗟。勞勞心跡鏡中花。一番春色覘天意。年運亨通卯最佳。


解曰

此籤先凶後吉。凡事逢春則吉。遇卯年月日。百事漸進。

若貪求。徒勞心力。求名晚成。訟可解。婚宜審。

孕要保。行人阻。謀望皆要平心和氣。方免是非也。


釋義

言百事營求。不但勞力。抑且費財。

奔走南北。終是無益。卻宜安坐。方可無悔。

待卯年月日。方得名利成就。如枯木逢春。發生暢茂。

凡事在春大亨。他季不吉也。


占驗

一生自南庠改北鹽。屢試不第。占此。自謂卯科當中京。

不驗後。值酉年始領鄉麻。乃悟兩座師一房。

俱是卯年所生。所謂玉兔交也。

又一人有兒子。(清)光緒。庚寅春月。患痘頗驗。

事完後再到廟。請示疹期。以便預防。占此。

果於辛卯年二月出疹。是玉兔交時。當得與再逢春。數字之應也。



故事

(一)薛仁貴投軍

唐。薛禮。字仁貴。龍門人。應募投軍征遼。白衣陷陣。

立奇功。太宗曰。朕不喜得遠東。喜得將軍。

累官左武衛將軍。封河南縣。

男當拒突厥於天山。發三箭射殺三人。突厥懼遁。

軍中歌曰。將軍二箭定天山。壯士長歌入漢關。


(二)張騫泛牛斗

漢。張騫。城固人。武帝時。奉命使大月氏。開西南夷。

嘗乘槎。直上天河。入斗牛宮。見一女子。手弄金梭。

傍有一石。問之不答。惟云可問嚴君平便知。

君平蜀人。精天文術數。往詢之。云此天孫(仙)織女也。石名支磯石。

（雷雨師一百籤第二十四籤亦有此故事）
