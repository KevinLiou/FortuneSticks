雷雨師一百籤


第五三籤【下下。己丙】

艱難險阻路蹊蹺
南鳥孤飛依北巢
今日貴人曾識面
相逢卻在夏秋交




聖意

病與訟。皆不利。名與財。亦阻滯。孕有驚。行未至。若遇貴。事方濟。


東坡解

艱難歷遍。作事皆非。南鳥北巢。失其所依。夏秋之交。貴人相提。應稱謀望。榮達可期。

 


碧仙註一

謀為先阻後通亨。事用心機遂所行。若得貴人提拔處。諸般方可事圓成。


碧仙註二

謀為先阻後還亨。事際心機遂所行。若得貴人提拔處。諸般方可此圓成。


解曰

此籤歷涉艱難。凡事不遂。如南鳥依北巢。失其所居之處。

若遇秋之後。方得貴人提攜。

病宜防。訟亦險。名利俱阻。行人未回。婚能成。財緩有。

言目下終未便也。


釋義

玄德贅孫堅女。起臥處。悉羅列劍戟。其歷艱險。

恍如失勢之龍。是南鳥依北巢。不得其所也。

後竄南歸。起諸葛。用關張。是得貴人識面也。

占者若所依之地。因其人而用之不得其宜。則是非紛紜。

動輒見咎。凡事夏末秋初。方遇貴人提攜。有泰來之兆。

名利遂意。但恐其終不如其始。

婚不易成。訟不得勝。他事慎之。


占驗

宋。紹興中。閩人。施宜生。陷賊中。百計得脫避。淮上占此。

人金獻策。金主擢居上第。累官尚書。左丞以出使。

至宋泄漏軍情。獲罪。此南鳥北巢之驗。

又(清)同治。戊辰。余仝五邑諸芸友。占此。

後惟南安楊生。坐東號舍見售。亦南鳥孤飛之應也。

竹坪識。



故事

劉玄德入贅孫權妹

劉玄德。即西蜀先主也。為荊州牧時。甘夫人(劉備妻)新喪。

東吳孫權聞知。詐以妹招贅欲賺(軟禁)其人。吳素討荊州。

孔明料知。乃授趙雲錦囊計。保先主入吳。(東江招親)

意弄假成真。娶孫夫人以歸焉。

有曰

周郎妙計高天下。賠了夫人又折兵。
