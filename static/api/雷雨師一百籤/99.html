雷雨師一百籤


第九九籤【上上。癸壬】

貴人遭遇水雲鄉
冷淡交情滋味長
黃閣開時延故客
驊騮應得驟康莊
(黃閣。指丞相公廨。漢公孫宏有開閣延賓故事)




聖意一

名與利。訟和事。家道康。皆吉利。

病即安。孕生驥。婚則成。行人至。


聖意二

名與財。皆得意。家道康。皆吉利。病即安。孕生貴。訟則伸。行人至。


東坡解

憶昔貴人。忽見邂逅。從此提攜。命運亨泰。

出入圖謀。凡事利快。士庶占之。前程遠大。


碧仙註一

如鴻遇順風。眾隨下水中。

得人輕助力。任意過西東。


碧仙註二

結交通義勝黃金。黃金不及至人心。王陽貢禹當年事。相慶彈冠待好音。


解曰

此籤上吉。家宅安穩。風水利。名與利皆快便。

婚姻成。行人至。病安全。孕生子。保平安。凡事皆吉利也。


釋義

光武與嚴光同學。厥後光武物色訪求。

於富春山安車迎之。延之黃關。

士庶占之。必得貴人提攜。但未實有富貴。

凡事主午上大利。

驊騮。良馬也。驟康莊。通且達矣。

水雲鄉如黃牛峽。白馬江。東南澤國之類。

遣遇于此。宜謹修為可也。


占驗

一人選時占此。後得武進。乃東南水澤之國。自甘冷淡。冀得久任。

又一人登科。占此久不第。直至座師拜相作總裁時。始第應第三四句。

又一人習賈。有才扶不得主。料其必敗。占此欲應海口。世好蘇氏聘。不果。乃為鳳閣街黃姓當事。將週歲。甫開市。而蘇氏以厚利爭向。遂於通衢。建一大生意。歷二十年號。已馳名。其人始卒。

此占連題義畢應。善解者。毋煩多贅焉。



故事

百里奚投秦

百里奚。宛人。初事虞公。晉假途滅虢。知其將亡。

不諫而先去之。秦復為勝。逃隱於楚。

秦穆公聞其賢。以五千皮向楚贖回。授以國政。

秦自強盛。并國二十。遂霸西戎。國人稱奚為五羖大夫。
