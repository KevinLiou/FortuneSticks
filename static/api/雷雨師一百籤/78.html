雷雨師一百籤


第七八籤【下下。辛辛】

家道豊腴自飽溫
也須肚裏立乾坤
財多害己君當省
福有胚胎禍有門
（豊字，另有改作豐字一說）




聖意

莫貪財。能害己。休闕訟。當知止。

病禱神。孕生子。婚擇良。行未至。


東坡解一

家道昌盛。猶宜謹守。自立主張。祈求獲佑。

輕財重道。方能長久。善善惡惡。自作自受。


東坡解二

家道昌盛。猶宜謹守。自立主張。祈求護佑。

輕財重道。方能長久。善善惡惡。自作自受。


碧仙註一

一人會處一人事。莫管他人閑是非。

若能作福求安靜。雖否還能轉福機。


碧仙註二

自家心地自家知。莫管他人閑是非。若能作福求安靜。雖禍終能轉福機。


解曰

此籤家道雖然昌大。亦宜謹守。不可倚強欺弱。

主有口舌是非。財多害己。不可妄求。

只宜集善去惡。財莫貪。訟亦休。問婚不宜。

行未回。病求神。休妄想也。


釋義

肚裏乾坤謂自主張也。可行則行。可止則止。一事三思。切勿貪戀。

胚胎有門。言一善一惡。俱是人之所為。

功名有分。終必不失。目下求財不利。尤宜謹省。


占驗

(宋)天啟時。一臺諫不順閹宦。恐有他禍。

占此乃休官散財。安居林下。後權閹勢敗。

此公仍復趣擢。此自明禍福之驗。

又一人。春月。妻病寢食不安。子又患痘。占此。

莫識所指。有為解曰。妻病無妨。肚裏有孕也。

子出天花。禍不遠矣。而渠云。

天癸纔去。廿餘日焉。有胎于出痘。並未危。

奚能死。均信。不數日。其子竟死。

至冬間。妻舉兩子。一男一女。其人始服二句與四句之驗如此。



故事

(一)袁安守困

漢。袁安。字邵公。汝南人。少時貧困。志行清高。

嘗高臥雪上。後室孝廉。除陰平長。任城會所在。

使人畏而愛之。至為三分時。漢舉中微。外戚強盛。

朝廷之上。皆倚賴安一人焉。


(二)石崇錦絲步帳

晉武帝的時候，有一個人名叫石崇，他官封安陽卿候，作到荊州刺史，他竟然派人暗地裡去劫掠航海的商販，因此而得到不少不義之財。他這個人只知道自己發財。從不為人民打算。平常他專門以和別人爭強鬪勝，來滿足自己的虛榮心。


當時朝廷皇親國戚王愷、羊琇等人，也都非常豪華奢侈。王愷作絲步帳四十裡，石崇就作錦步帳五十里。有一天皇帝賜給王一棵三尺高的珊瑚樹。王愷就拿出來和石崇比較石崇隨手那起一隻鐵如意，用力一擊，把珊瑚樹打的粉碎。王愷正準備和他理論，石崇命下人把他所有的珊瑚樹，全部搬了出來，其中有六、七棵，竟高達四尺多。像王愷那樣三尺高的真是不計其數。石崇隨便拿了一棵賠給王愷。


經過這次事件，再也沒有人敢自不量力，和石崇來比財寶了。當然石崇心裡沾沾自喜，更加的傲慢而不可一世了。


石崇以他無比的財力，在安陽地方建築了一所金谷別墅。從重樓疊閣，畫棟彫樑，真是金堆玉砌，說不盡的富麗堂皇。他擁有的姬妾使女數以百計，個個都穿戴錦繡珠翠，服飾華麗，豔光照人。絲竹管絃，音樂之聲，終日不絕於耳。


那時有一位叫賈謐的朝廷權貴，因賈后事件的牽連，被趙王所殺。石崇因為也有附和逆黨的嫌疑，被革去官職。


石崇有一位寵愛的侍妾名叫路綠珠，長得非常的美豔動人，可以說是國色天香。有一天被趙王的心腹孫秀見到了，驚為天人，於是就派人去向石崇示意，希望石崇能夠割愛，把綠珠送給他。


但是綠珠是石崇最寵愛的女人，他怎麼可能捨得讓他投入別人的懷抱，因此一口回絕了。孫秀懷恨在心，立誓要報復，就找了個機會向趙王進讒言，陷害石崇，硬説石崇有叛國造反的意圖。趙王不察，就下詔判石崇死罪。


當石崇被押赴刑場，走在半路上的時候，他不禁感慨的高聲叫著：「你們這些奴才，想要霸佔我的家財，竟然不顧道德來誣陷我，要把我害死！」


旁邊監刑的官史聽覺了，卻冷冷的，反問他：「你既然知道多足以招禍，當初何必要非法聚歛呢？家財既然富足了，又何不趁早疏散，多做點施貧濟困的善事，積積陰德呢？」

他聽了啞口無言。他的全家同時遭到誅連，龐大的財產也全部被充公了。

石崇的病是貪婪驕奢，他的財富是由搶劫，並且殺了不少無辜商人而得來的。所謂劫財越貨，與強盜沒有兩樣。古人說：「多行不義必自斃」結果滿貫家財免不了被人家奪回去，還賠上了一家老小的性命！可見天理昭彰，報應不爽。


提醒您

台灣的電視曾經播映過的黃香蓮歌仔戲『綠珠樓』就是石崇和路綠珠的故事。
