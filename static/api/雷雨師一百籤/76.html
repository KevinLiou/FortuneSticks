雷雨師一百籤


第七六籤【中平。辛己】

三千法律八千文
此事何如說與君
善惡兩途君自作
一生禍福此中分




聖意一

問公訟。且審理。求財祿。當揣己。

病蚤禳。宜求嗣。婚更審。方吉利。


聖意二

問爭訟。且斷理。求財祿。當揣己。病早禳。勢莫使。婚更審。方吉利。


東坡解

吉凶多變。非止一端。言不盡意。當以心觀。

福因積善。禍胎稔惡。知所趨避。宜自先覺。


碧仙註一

訟庭有險莫求尋。只恐憂疑亂了心。

凡事要知前定數。但將善惡自浮沉。


碧仙註二

訟庭有險莫胡行。無是無非夢不驚。禍福原來無定數。權衡善惡自分明。


解曰

此籤謀事。陰人議論不盡。當以心觀。積善福來。

積惡禍胎。修善避惡。宜自先覺。

婚宜審擇。財未遂。病不防。訟主凶。宜作福。方吉。


釋義

言吉凶通塞。當遂所占之事。以理推之。

如作善降祥。作不善降殃。所謂此中分也。

求功名者。初未遂。後顯擢。

凡事營謀。有二條路。占者宜審決於善趨惡避。

切莫貪取躁進。大抵先憂後喜也。


占驗

一生人場畢。自謂文字得意。占此竟無名。後收原卷觀之。

前場密圈擬元。後場則塗抹不堪矣。此三四兩句之驗也。

又余姑丈。龔慎齊。明經當廩保時。占財氣。得此。

時府試挨保無贄。敬惟院試每人送百文計。文武塗并進。

一生三款。合算得銅錢八千文。籤經之明示。其驗如此。

又(清)光緒。己卯。泉州南臺寺。文會晉江。

邱生帶籤普。在大魁殿用錢卜此。後鄉闈。惟邱生卷八次到取中。

此亦八千文。及與君數字之驗焉。

王錢謹識。



故事

(一)蕭何註律

漢。蕭何。沛郡。豊人。微時。嘗為刀筆史。

後佐漢高帝定天下。為宰相。以功封酇侯。

漢既一統。以三章之約。非可長治天下。

乃命蕭何註律。使民知所守法焉。
