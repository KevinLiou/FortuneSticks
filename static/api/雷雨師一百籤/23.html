雷雨師一百籤


第二三籤【下下。丙丙】

花開花謝在春風
貴賤窮通百歲中
羨子榮華今已矣
到頭萬事總成空




聖意一

名與利。似虛花。訟解散。病益加。

婚未合。行人賒。事無就。徒咨嗟。


聖意二

名與利。似虛花。訟解散。病愈加。

婚未合。行人賒。事無就。徒咨嗟。


東坡解一

凡百謀望。有成有敗。始雖快意。終實成害。

吉中有凶。否生於泰。若不歛藏。後憂方大。


東坡解二

凡百謀望。有成有敗。始雖快意。終竟成害。

吉中有凶。否生於泰。若不歛藏。後憂方大。


碧仙註一

此籤大不中。花開遇烈風。

不利老人占。病訟盡皆凶。


碧仙註二

此籤大不終。花開遇暴風。榮華全剝落。病訟盡皆凶。


解曰

此籤求望有成有敗。初快意。終有害。

吉中有凶。否有泰。不善歛藏。後憂更大。

名利虐。訟宜散。病防老人為凶。

婚未成。行人遲。失物之憂。大不吉也。


釋義

富貴窮通。四者循環。得失不久。如花之開落無時也。

今已矣者。言凡事泰必復否。否必復泰。乃數之極。變之始也。

但占者不為泰所移。不為否所困。斯為人定亦能勝天。

病防老者。路防失財。婚防有別。孕必生女。


占驗

一生未遇時。占此自謂。無發達之日。

後入場坐空字號。遂得中榜。應在總成空之句。聖詩微妙。人不易測。

又泉城內。郭生有子得病。延兩醫。一名榮觀。一名今先。

因寒熱爭辯。遂入廟占此。猜者謂今已矣。決不合。

勸其托榮觀調治。至年終。果愈。此又羨子榮華到頭總成空之驗矣。



故事（兩個故事皆可參考）

(一)吳王愛西施

吳既滅越。越王勾踐。卑詞請成。臥薪嘗膽。果報會稽之仇。

與大夫范蠡。文種謀。乃選民間絕色西施女。進之吳王。

夫差甚嬖愛之。至於亡國。


(二)殷浩書空

殷浩，字深源，晉陳郡長平人。識度清遠，好老易，負有盛名。初為庾亮記室參軍，後起為建武將軍。朝廷欲平關河，以浩為中軍將軍，都督揚、豫、徐、兗、青五州軍事，以平定中原為己任。征姚襄兵敗，廢為庶人，竟日書空，作咄咄怪事四字。（引用中華人網站資料）

 

提醒您

本籤寓意當初玩夠了，現在玩完了。這就是命理師俗話說的『運走完了』的意思。
