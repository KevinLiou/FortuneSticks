雷雨師一百籤


第四三籤【中吉。戊丙】

一紙官書火急催
扁舟速下浪如雷
雖然目下多驚險
保汝平安去復回




聖意一

功名遂。子嗣歡。訟病險。終必安。

失物在。行人還。婚宜遠。利不難。


聖意二

功名遂。好求官。訟病險。終必安。

失物在。行人還。婚宜遠。利不難。


東坡解一

官書火急。激聒之兆。目下雖驚。終無紛擾。

有神陰扶。收功談笑。來往平安。吉星高照。


東坡解二

官書火急。激聒之兆。目下雖驚。終無紛擾。

有神陰扶。收功談笑。來往平安。吉星拱照。


碧仙註

先驚未是驚。後喜方為喜。

若問謀事多。連綿無定止。


解曰

此籤先凶後吉者。謀望終有好處。

病者虛險。自然解散。大事成。求官遂。問訟好。

婚姻不難。失物還。行人至。凡事大吉。


釋義

火急催者。王有激聒。是非占者。目下驚恐。到後無事。

新病易廖。久病難保。凶事化吉。大事變小。

求名利者。有疾病是非之累。方應其占。

而得選舉。否則難遂。

回字或作來字。


占驗

一生會試。值以人命搆訟所阻。占得此後。

司憲見而奇之。許令應試。遂中。春榜全詩皆應。

此占。凡問病無有不廖者。惟(清)道光(年)間。一人患中瘋症。

占此後。雖平安。但病根終不能除耳。是去復回。句之應也。



故事

(一)玄德公黃鶴樓赴宴

漢。劉備。字玄德。獻帝時。群雄割據。曹操挾天子令諸侯。

玄德未得四川。借與荊州。托足周瑜。計請赴宴。設伏挾討。

幸孔明授趙雲密計。計得免於難。此係俗傳。

攷史。赤壁合兵拒操時。瑜計請玄德。設伏欲害。

因見關公在場。不敢動手。


(二)陳平亡歸漢

陳平，初事項羽，后知羽必敗，乃仗度河歸漢....

陳平，漢初陽武人。少時家貧，與兄同居，有薄田卅畝。他酷愛讀書，好黃老之術。陳勝、吳廣大澤鄉起義，他投靠魏王咎，為太僕。後從項羽入關，任都尉。旋歸漢王劉邦，任護軍中尉，屢以奇計佐劉邦平定天下。漢朝建立，封曲逆侯。惠帝、呂后時任丞相，以呂氏專權，不治事。呂后死，他與周勃定計翦滅諸呂，迎立文帝，任丞相。


陳平終生銳意進取，多出急智，頗具縱橫家的風采，是中國古代很有影響的謀略家。
