雷雨師一百籤


第四四籤【中平(中吉)。戊丁】

汝是人中最吉人
誤為誤作損精神
堅牢一念酬香願
富貴榮華萃汝身




聖意

事多錯。訟莫作。病禱神。且勿藥。

婚莫求。行未還。能自悔。利名全。


東坡解

心行平等。反誤為非。急宜改過。切莫自欺。

盡誠作福。神必佑之。榮華富貴。付汝非遲。


碧仙註一

萬事皆從忙裏錯。但將心地細思量。

若能謹守無他望。可保終身大吉昌。


碧仙註二

萬事皆從忙裏錯。但將心地細思量。

若能謹守祈神佑。可保終身大吉昌。


解曰

此籤防有橫逆事。若能改過。則吉。

凡事且也不宜躁進。訟莫作。利未有。婚莫求。病禱神。

行人未歸。悔過名利雙全也。


釋義

吉人善人也。言人雖善。作事狂謬。所輔非人。作為皆誤。

但能堅心向善。謹而行之。則富貴可得。終身榮華矣。

占者不宜欺妄輕躁。先自知戒。可免其悔。

求財無利。求名未亨。

萃字或作在字。


占驗

一生本富人子。性傲不敬其父。屢中副榜。占得此籤。

改悔行孝。父子俱安。享壽以終。

倘不能改。豈惟不中。禍將不免矣。

全詩之意。顯然能改過勝酬香願矣。

又咸豐間。郡西華仕舖。端許節。

土著循俗。例迎木刻龍頭。與東佛舖民口角。

邑尊聞知。飾差嚴拿該舖。惟奇仕官香火甚盛。

老尼有錢怕累。因到廟占此。迨事情完局。

富貴之家。分文不肯看破。衙行賠禮等費。

悉萃老之身。第四句之驗如此。



故事

(一)王莽篡漢

王莽字巨君。漢。孝元皇后之姪也。以國戚一門貴顯。

莽繼叔父遺烈當成哀平帝。享國不永。統嗣三絕。位居宰相。

封新息侯。假行仁義。自此伊周弒平帝。立孺子嬰。

竊周公之法。攝位改元。竟移漢祚。國號曰新。

後被漢兵所殺。傳首師百姓。爭食其肉。


(二)留侯博浪椎

張良其祖先是戰國時韓國人，五代在韓為相。秦國滅韓後，張良散家財而求得一大力士相助，用一百二十斤的大鐵椎伏擊行刺秦始皇於博浪沙（今河南原陽縣）中，卻擊中副車而未獲成功。
