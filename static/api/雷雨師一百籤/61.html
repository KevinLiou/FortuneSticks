雷雨師一百籤


第六一籤【中平(中吉)。庚甲】

嘯聚山林兇惡儔
善良無事苦煎憂
主人大笑出門去
不用干戈盜賊休




聖意

財平平。病漸效。訟自散。莫與較。

遠行歸。婚亦宜。雖有險。終平夷。


東坡解一

惡少操心。欲害良善。彼何能為。徒逞強健。

君子善勝。收功不戰。有神相之。陰消禍變。


東坡解二

惡少操戈。欲害良善。彼何能為。徒逞強健。君子善勝。收功不戰。神明相之。陰消禍變。


碧仙註一

心驚如踏一層冰。長敬番成度量生。一葦一航波浪足。駸駸喜色藹門庭。


碧仙註二

心驚如踏一層冰。戒懼何憂險阻深。一葦一航波浪定。駸駸喜色靄門庭。


解曰

此籤主有驚險。防小人侵害。若能守性存善。卻有德曜及身。

神明助之。陰消禍變。病恐險。訟必散。財平平。

婚宜審。凡事有陰。及後平和。


釋義

灌嬰將兵擊尉陀。人民驚恐。神在南越山上。

告將軍曰。干戈不動。而盜賊遽休。後果奏凱。

占此主小人侵害。事多驚恐。若能修福。則惡不能侵害。

危病小口(心)無事。家主見凶。

凡事先憂後喜。名利險中可得。


占驗

一生試事畢。卜捷音。占此後。不得中榜。則此籤大不利於求名也。



故事

(兩個故事都與籤文關聯不大)


(一)蒯輒見韓信

漢。蒯輒。字通。范陽人。時韓信假節為齊王。

輒知天下權在韓信。往說之曰。今大王威震天下。

與楚則楚勝。與漢則漢勝。大王若不楚不漢。

鼎足三分。可長保富貴。

信謝之曰。吾受漢王厚恩。終身不忍背漢。

後信被呂后所殺。臨刑有悔不聽蒯通之言。


(二)祭征虜雅歌投壺

祭征虜，是西漢光武帝時的征虜將軍祭遵，《後漢演義第十八回》記有：『遵為人廉約小心，克己奉公，所得賞賜，盡給士卒，家無私財，身無華服，取士專用儒術，對酒設樂，必雅歌投壺，饒有儒將風規。』


《後漢書‧祭遵傳》，“遵為人廉約小心，克己奉公。賞賜輒盡與士卒，家無私財。”是成語『克己奉公』的來源。


《后漢書卷二十‧銚期王霸祭遵列傳‧第十》也有『祭遵好禮，臨戎雅歌』的記載。祭遵出身豪門，但生活簡樸知書達禮，且執法嚴明不徇私情。


提醒您

以上兩個故事都和籤文關聯不大，解籤詩可依籤文與兩個故事三者擇一最適者求解。
