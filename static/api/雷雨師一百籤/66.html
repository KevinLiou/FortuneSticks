雷雨師一百籤


第六六籤【上上。庚己】

耕耘只可在鄉邦
何用求謀向外方
見說今年新運好
門闌喜氣事雙雙
(見說二字，另有改作只見一說)




聖意一

病即安。訟即決。財漸豊。名高揭。

行人回。婚可結。莫外求。福儘得。


聖意二

病即安。訟即決。財漸豐。名高揭。

行人回。婚可結。莫外求。福儘得。


東坡解

出外不宜。在家必達。時運將亨。財祿俱發。

好事重重。門庭改革。若欲遠圖。弄巧成拙。


碧仙註一

自家安分莫他求。自有榮華切莫憂。

喜得一陽天氣象。也知春煖勝三秋。


碧仙註二

自家安分莫他求。俯仰寬舒得自由。

現在陽和春已到。桃紅李白樂悠悠。


解曰

此籤不宜遠行。只宜守舊。自有超達。時運漸亨。

財祿俱發。訟不宜。財自至。婚可求。名利高。

病獲安。孕生貴子。風水利順。家宅昌隆也。


釋義

名利只宜近處。不宜遠行。守舊好。進步難見。

說新運好者。不是決。許其好。尚當待時發達也。

占此者。凡事皆利。且得護喜。婚姻事尤宜。

耕耘或作經營。向字或作出字。


占驗

正德間。京師揚惟聰占此。後以鼎元歷官翰苑。

至宮保學士尚書。凡十餘任。並不出京。正應首句。



故事

(一)杜甫遊春

唐。杜甫。字子美。杜陵人。

玄宗朝獻賦。命待詔集賢院。後轉檢校工部員外。

卻甫覽君書。善為詩歌。涵渾汪洋。千態萬狀。

陳時事。確切精深。世號詩史。

元禎謂。詩人以來。未有如子美者。

惟好賞春山。故每當春晴。必遊郊外。


(二)諸葛隱南陽

必如諸葛孔明之隱處南陽，不屑輕出，待至劉備三顧，勤勤懇懇，方效驅馳，名士之出處，如此慎重，豈田疇輩所得望具項背乎？三國人才眾矣，如孔明者，其固超類軼群哉！

籤詩網研究室

劉備有沒有拜訪三次，有些小心眼的人一直在懷疑，其實應該有。

《前出師表》臣本布衣，躬耕於南陽，苟全性命於亂世，不求聞達於諸侯。先帝不以臣卑鄙，猥自枉屈，三顧臣於草廬之中，諮臣以當世之事；由是感激，遂許先帝以驅馳。

前出師表目前大多不認為是偽作，『表』是呈給君王的，不是茶館裡說書用的，沒有『三顧』而寫下『三顧』，這可是欺君。如果孔明認知劉備實際上沒有去三次，而故意寫三次的話，那是孔明

欺君外加自抬身價，偏偏歷史上不會同意這樣的人格評價，所以基本上不用懷疑『三』這個數字，不多不少，三次就是三次（至少孔明相信是三次）。


當然也有一種可能，就是當初劉備自己告訴孔明先前已經去過兩次，那就......，不過這純屬我們以小人之心度君子之腹，至此看來似乎好像連我們也變得很小心眼了。
