雷雨師一百籤


第九十籤【中平。壬癸】

崆峒城裏事如麻
無事如君有幾家
勸汝不須勤致禱
徒勞生事苦咨嘆

(勞字，另有改作營字一說)
(嘆字，另有改作嗟字一說)




聖意一

訟和吉。求財無。婚未成。病無虞。

信未至。勿他圖。不須禱。守規模。


聖意二

訟和吉。求財無。婚未成。病無虞。信未至。勿他圖。不須禱。且安居。


東坡解一

人事擾擾。君獨清閒。不必瀆神。妄求多般。

反招口舌。惹起禍端。急宜靜守。以保平安。


東坡解二

人事擾擾。君獨清閒。不必瀆神。妄求多般。反招口舌。激起禍端。急宜靜守。以保平安。


碧仙註一

在家安分好。莫笑別人難。

若是貪憎痴。身遭羅網間。


碧仙註二

閒人厭安閒。妄想忽多般。惹起無干事。求閒卻是難。


解曰

此籤謀望事艱難。家門招禍愆。訟必和為貴。慎勿聽人言。

求財決定失。問病也無安。懷孕宜早保。行人無信還。

若問婚姻事。渾如過險山。


釋義

言似有事而實無事。占者但當主靜。不可貪圖。

雖勤致禱。終無所益。謂之曰。禱者以其邀福也。

殊不知。福降於作善之人。若去妄求。則禍隨時而至矣。

崆峒在高平縣。凡城居之家。必然多事。

豈特崆峒為然。亦泛言之耳。占者詳之。


占驗

一人訟事占之竟無相干應第二句也。



故事

(一)楊文廣陷柳州

正史稱。宋。楊文廣。延昭子。從狄青南征。

儂智高為廣西鈐轄。兵陷柳州。青救之。

與野史云。文廣係宗之子。陷柳州被困。楊家諸女將往救焉。

傳術迥異。姑併錄之。仍宜從正史為是。


(二)塞翁失馬為福（出處《淮南子．人間訓》）

古時候北方的邊塞有一位善於養馬的老人，大家都叫他做塞翁。有一天，他養的一匹馬偷偷地從馬棚裏逃到胡地去了。鄰居們知道以後，都走來安慰塞翁，勸他不要難過。可是塞翁卻一點兒也不傷心，還笑著說：『馬雖然走失了，說不定能給我帶來好處呢。』

過了幾個月，那匹馬跑回來了，而且還帶來一匹胡地的駿馬。鄰居們知道後，又走來向塞翁道賀，塞翁卻皺著眉說：『白白得來一匹駿馬，說不定會帶來災禍呢。』

塞翁的兒子喜歡騎馬，有一天他騎著那匹得來的駿馬遊玩時，竟然被駿馬從背上摔了下來，把腿也摔斷了。鄰居們聽說這件事之後，紛紛趕來慰問塞翁。塞翁並不十分傷心，淡淡地說：『我的兒子腿斷了，說不定|因禍得福呢！』鄰居們都莫名其妙，心想塞翁定是傷心過度，腦筋糊塗了。

不久，胡人大舉入侵邊疆，所有青年男子都被徵調當兵，由於胡人兇悍，很多當兵的青年都戰死沙場。塞翁的兒子因為斷了腿，所以不用當兵，總算保存了性命。這時鄰居們才明白到塞翁的智慧。
