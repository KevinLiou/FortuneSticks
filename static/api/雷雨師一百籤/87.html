雷雨師一百籤


第八七籤【下下。壬庚】

陰裏詳看怪爾曹
舟中敵圖笑中刀
藩籬剖破渾無事
一種天生惜羽毛
(圖字，另有改作國字一說)
(羽字，另有改作鳳字一說)




聖意一

名利無。病有祟。訟莫興。和為貴。

莫貪財。婚不利。孕無憂。行未至。


東坡解一

骨肉乖張。操戈入室。面上春風。胸中荊棘。

何不思維。人己則一。若能和同。變凶為吉。


碧仙註一

自家何必操弋戟。若有平生便釋然。

所在所為天地見。陰消漸弭可無愆。


碧仙註二

白日青天大丈夫。何須猜忌暗相圖。

推心置腹堪唇齒。交濟成功兩不殊。


解曰

此籤先凶後吉。凡事要和同。可免是非。

病有祟。未安。訟傷手足之親。只宜和解。

防暗箭。求財遂。卻被相熟之人耗散。

婚有變。不合行人。未即至也。


釋義

言人受形父母。初無少異。譬如桃仁。則生桃枝。亦不變種而生。

故占者當慎惜鳥毛。養成飛騰之具。則去貧賤而受富貴矣。

占此主骨肉猜忌。小人侵害。苟能大公無我。

取信於人。則自可免。尚宜慎之。


占驗

一生攜重資赴試出行。占此。登舟遇一人。求附舟。

失足落水。衣盡濕。此生解衣贈之。復與酒食溫煖之。

臨別謂生曰。我盜也。將有不利於君。感君之義。故謝去耳。

生復厚贈之。雖此生之德感化其心。亦籤經之明示昭然也。



故事

武侯與子敬同舟

曹操已破劉備。備奔夏口。依劉琦。

操乘勢下江南。孫權大懼。和戰未決。

聞曹兵聲勢百萬。乃使魯肅往夏口探問虛實。

並且孔明同入吳肅。欲戰於舟中。

囑孔明曰。見吾主切勿言曹兵之多。勸其決戰。

合力拒曹。願先生相助焉。
