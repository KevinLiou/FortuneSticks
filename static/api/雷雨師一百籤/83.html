雷雨師一百籤


第八三籤【下下。壬丙】

隨分堂前赴粥饘
何須妄想苦憂煎
主張門戶誠難事
百歲安閒得幾年




聖意一

名與利。難遽致。訟宜和。行未至。病瘥遲。婚莫議。能守待。家必利。


聖意二

名與利。莫貪求。訟宜和。行未至。病瘥遲。婚莫議。能保守。家必利。


東坡解

好事隨緣。莫貪莫競。且宜守成。庶保安靜。若更妄求。必主多憂。何如退省。以遂優游。


碧仙註一

守己當知事莫為。莫將心術肆凌欺。依然舊日規模好。休聽閒人說是非。


碧仙註二

貧賤安常識者希。天時人事貴知幾。

依然舊日規模好。休聽閒人說是非。


解曰一

此籤宜守成。不可妄為。凡事且隨緣分。不宜出行。

婚未成。財莫貪。名利休問。訟宜和。病未痊。六甲保。

墳宅改移。安分謹守持家。方吉利也。


解曰二

宜隨分退守。恰得安常處順之樂。若貪現前而圖意外。不唯無益。更曾煩惱而已。雖則家計艱難。然安閒境地。不可多得。合不愛惜光陰。甘守清況之為得乎。凡事恬靜則吉。妄動則凶。（臺北行天宮）


釋義

大抵妄想則貪求。貪求則取禍。

人生百歲。其中無禍無虞。安亨和平之福者。能有幾時。

占者當退卻一步。坐食饘粥以待時至。

主張門戶誠難事者。蓋言繼述無人。所以不得相助有成也。


占驗

一生小考。占此。發案列優等補廩。應赴饘粥也。

又一大剎。僧欲謀為住持。占此遂止。

後見主僧操持勞苦。迎送干謁煩惱。怨恨不好。

己之安閒清靜。始悟籤言不謬。

又余妹倩。按照磨林名樹立之。母病危。占此獲痊。應百歲安閒得也。

又一人。被人討欠。討欠人債迫太急。欲賣屋償之。

占此竟賣不得。後果就收入。還出了時事。方悟二三句之驗如此。

竹坪王錢識



故事

(一)諸葛孔明學道

漢。諸葛亮。字孔明。瑯琊人。漢末避居南陽。

志學清修。養生之道。研究奇明(門)遁法。天文術數。

卦算韜略。無所不精。初不求聞達於諸侯。

後徐庶薦於劉先生。先主凡三往問計。延為軍師。

先主即位。拜丞相。受以託孤。後主封為武卿侯。卒諡忠武。


(二)邵君平卜肆垂簾（應為嚴君平）

嚴遵賣卜於成都市，日得百錢自給，則閉肆下簾。富人羅沖為具車馬衣糧，君平曰：「吾非不足也，子奈何以不足而有餘？」沖曰：「吾有萬金，子無儋石，何云有餘？」君平曰：「不然。吾宿子家，人定而役未息，晝夜汲汲，未嘗有足。今我以卜為業，不下床而錢自至，猶餘數百，不知所用。此非我有餘而子不足乎？」君平嘗嘆曰：「益我貨者損我神，生我名者殺我身。」


向子平讀易，至損益卦，喟然嘆曰：「吾已知富不如貧，貴不如賤，但不知死何如生耳？」嫁娶畢，敕斷家事，云：「當如我已死。」與同好秦子夏俱遊五嶽名山，不知所終。


提醒您

明明是嚴君平（嚴遵）的故事，故事卻標為『邵君平卜肆垂簾』，我們推測或許是當初有人將梅花易數的邵雍與嚴君平混淆所致。
