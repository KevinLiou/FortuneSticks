雷雨師一百籤


第六九籤【下下(中平)。庚壬】

捨舟遵路總相宜
慎勿嬉遊逐貴兒
一夜樽前兄與弟
明朝仇敵又相隨
(樽指酒杯)




聖意一

名與利。莫強求。醫宜審。婚難謀。

行人至。訟可休。凡出入。謹交遊。


聖意二

名與利。莫強求。醫用審。婚難謀。

行人至。訟可休。凡出入。謹交遊。


東坡解一

捨舟登路。宜踏實地。莫伴貴兒。自當年記。

好中成惡。口舌立至。戒之慎之。樽節遊戲。


東坡解二

捨舟登路。宜踏實地。莫伴貴兒。事當牢記。

好中成惡。口舌立至。戒之慎之。樽節遊戲。


碧仙註一

得意濃時終失意。親朋遇我莫相隨。強言說得如甘蜜。那得開心見膽時。


碧仙註二

得意濃時終失意。浮華隊裏莫相隨。翻雲覆雨炎涼態。那得開心見膽時。


解曰

此籤出入道路。宜踏實地。謹防親人侵害。

時酒遊戲。好中成惡。口舌立至。

訟者休。病審醫。終不利。行人至。

名與利。莫貪求。宜謹慎。省是非也。


釋義

言謀為。當擇交遊。凡事好中成惡。是處成非。

所謂樽前兄弟。明朝仇敵也。

若占名利。得此只宜安守。不可強求。

凡有出入。宜向善背惡。切勿輕忽。以貽後悔。

占孕生男。主有產難。慎之慎之。


占驗

楚中一士人。遠至晉地。干謁顯宦。

占得此籤後。果與公子笑言相席。謔前鬨鬧。其人忽悟籤語。

次早不辭而歸。怨者欲害之而不得。其驗如此。



故事

(一)孫龐鬥智

齊。孫臏。魏。龐涓。其初同事鬼谷子。

涓之才學弗如臏。嘗與鬥智輸之。涓忌其能。致結成仇。

彼此謀害。君子觀於異日。涓刖臏足。臏喪涓命。

互相殘忍。毋乃其不仁不智之甚與。


(二)蕭朱隙末

尚無法直接取得蕭朱隙末故事，但有成語『王貢彈冠』堪比照參考。

《教育部成語典『王貢彈冠』》

王，王吉，字子陽。貢，貢禹，王吉的好友。彈冠，彈去帽子上的灰塵，準備作官。王貢彈冠指王吉、貢禹相交友，王吉在朝中當官，友人貢禹便彈去帽上的灰塵，準備作官的故事。後泛指一人出仕，好友同慶。漢書˙卷七十八˙蕭望之傳：「少與陳咸、朱博為友，著聞當世。往者有王陽、貢公，故長安語曰：『蕭、朱結綬，王、貢彈冠』，言其相薦達也。」

《回天寶籙》

王貢彈冠，古事托之空論。

蕭朱結綬，隙末於以相尋。

摩肩之雅意罕逢，

刎頸之結交鮮見。
