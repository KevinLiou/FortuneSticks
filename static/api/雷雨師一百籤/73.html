雷雨師一百籤


第七三籤【下下。辛丙】

憶昔蘭房分半釵
而今忽把信音乖
痴心指望成連理
到底誰知事不諧
(昔字，另有改作惜字一說)




聖意一

名利無。訟休爭。婚事吹(吹本作娶)。孕多驚。病危險。命迍邅。宜作福。保安全。


聖意二

名利杳。訟終凶。婚重娶。孕成空。病危險。運欠通。事無准。鮮始終。


東坡解一

事多離散。難望再成。痴心未解。尚自客情。

縱能強合。終必敗盟。凡事謀望。守宇真誠。


東坡解二

事多離散。難望再成。痴心未解。尚有縈情。縱能強合。終必敗盟。凡事謀望。另改途程。


碧仙註一

訟事有阻。問財成空。

家人嗃嗃。憂心忡忡。


碧仙註二

婚姻莫問。財利落空。

家人嗃嗃。憂心忡忡。


解曰

此籤事多散。艱再合。雖強合。終不久長。

病險。訟不吉。功名虛喜。婚主重娶。終不到頭。

行人不至。孕驚恐。墳宅空虛也。


釋義

言婚不諧。訟者休問。名利始終更變。決難永保。

但人之痴心未絕。徒勞指望。凡事皆難。不止名利也。

孕不成。子難養。雖無鼓盆之怨。亦有陰人之損。

(註：鼓盆或指包公之烏盆案。謀財害命。搬屍窯洞。火化造盆)


占驗

一人為子擇媳。占此。不信籤意而強成之。

其子婚。不數月。出外經營。失意浪蕩不歸。

其媳堅志。苦守清節以終。其驗如此。

又(清)光緒間。晉江縣。有人為其妻問病。占此。竟斷絃。



故事

(一)王昭君憶漢帝

昭君。歸州人。漢元帝宮女。會匈奴求婚。昭君詣闕上書請行。

既嫁單于。其心未嘗忘漢。嘗上漢帝書曰。

臣妾幸得備禁臠。以為身依月日。死有餘芳。豈料失意丹青。

遠竄絕域。命也。奈何妾有父有弟。願君王憐而愛之。


(二)齊侯興師伐蔡

就戰略而言，齊桓公攻打蔡國的小戰爭不值得一提，但有趣的是這場戰爭的前因和後果。很難得看到有人將歷史故事描述得如此精彩，所以引用這一篇：

前言

　

在中國歷史上，男人掌握政權，在男人眼中，女人只是權力的附屬品，甚至是拿來賞賜的禮物，男人往往不知道女人在政治版圖中有著重要角色，女人的重要性在父權社會被大男人主義給掩蓋了，事實上，女人常常有左右政治局面的巨大力量，以下就是一個霸主因為一個女人而掀起了一場莫名其妙戰爭的故事。


本文


話說春秋五霸中的齊桓公，有一天跟他寵愛的夫人----蔡姬，心情愉悅地在花園中乘著小船遊玩，天真無邪的蔡姬忽然搖晃著船身，她覺得很有趣並咯咯地笑著。


沒想到齊桓公相當害怕，他很兇地跟蔡姬說：「不許再搖了!聽到沒？」沒想到蔡姬不當一回事，


「唉唷，不過搖個船有什麼好大驚小怪的呀？」


因此她又繼續搖，邊搖還邊說：「哈哈哈，原來主公你怕水呀!!」齊桓公惱羞成怒了，原來齊桓公小時候掉到水裡過，他被救起來的時候很狼狽，所以齊桓公因此而很怕水，不過這是他的小秘密，沒人知道，沒想到被蔡姬這麼一盪，他的小秘密就被發現了，而且還被大家講說「原來霸主也會怕水呀」，真的是很丟臉，再加上他找不到台階下，他就很生氣地把蔡姬送回娘家，齊桓公本意是希望她可以好好反省自己，並且跟自己道歉好讓自己有台階好下。


沒想到，蔡姬被送回娘家之後就跟爸爸撒嬌說：「把拔，你都不知道他被我在船上晃來晃去嚇得半死的樣子，真的好笨喔，而且竟然因為這樣就把我送回娘家了，我不要回去了啦!」蔡姬的爸爸想想也覺得女兒說的很對：「我這女婿不會游泳不能陪我女兒玩水就算了，竟然還罵我的寶貝女兒，真是太可惡了!」因此蔡


姬就在父親的安排下嫁給別人了，齊桓公知道了，氣到臉都綠了，他在心中吶喊：「竟然把我心愛的老婆嫁給別人，那不就等於幫我戴綠帽，而且我連下來的台階也都沒有了，氣死我了，蔡國的那隻老狐狸敢陰我，我一定要打蔡國，我要雪恥~~~恥~~~」


這時，齊桓公的宰相----管仲知道了齊桓公想要攻打蔡國的事，就說：「如果我們因為蔡國把蔡姬嫁掉而攻打蔡國，會讓大家笑話的。」齊桓公說：「這我管不了，這口氣我忍不下去!!」於是管仲只好幫齊桓公想出攻打蔡國之後再去攻打楚國的辦法，用以混淆國際視聽。


於是隔年春天，齊桓公就率領諸侯軍隊去攻打蔡國，蔡國一下就被打敗了，齊桓公接著將軍隊轉去攻打楚國。


楚國國君聽說齊桓公帶著軍隊來了，覺得很詭異，便出來與齊桓公說：「齊桓公呀，真是久仰大名，阿~~不過，你在北邊當霸主，阿~~我在南邊，我又沒惹到你，你幹嘛說來打我就來打我勒？」這時，管仲說：「以前召康公曾經授與我們齊國的祖先使命，要我們號招天下諸侯輔佐周王室，南到穆陵，北至無隸都是我們管轄的範圍，而你們楚國就在這個範圍內，卻沒有向周天子進貢苞茅，害周天子的祭祀時不夠恭敬，這是你第一個過錯；還有西周昭王到你們南邊之後卻再也沒有回來了，這是你們第二個過錯，我現在替周天子命令你把苞茅還有昭王給交出來，不然我將代替周天子懲罰你。」


楚國國君聽了覺得很好笑，便回答管仲：「喔，是這樣喔，阿~~沒有進貢苞茅是我的錯，我馬上把苞茅進貢給周天子，這樣好不好？那這第一個過錯應該就不算數了吧，至於你們昭王沒有回去，我是覺得你可能要去跟河神要人才對啦！阿~~是關我什麼事？」


結語


最後這場戰爭就在兩邊的叫囂聲中結束，也沒有什麼所謂的誰輸誰贏，反正齊國攻打楚國這場戰爭本身就是一場幌子，只是好笑的是，戰爭的原因只是因為齊桓公的老婆被嫁掉了，他覺得很羞辱要雪恥，只是這樣就可以發動一場戰爭，可見女人真的很厲害。
