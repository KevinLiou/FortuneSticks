雷雨師一百籤


第三七籤【中平(中吉)。丁庚】

焚香來告復何辭
善惡平分汝自知
屏卻昧公心裏事
出門無礙是通時




聖意一

訟和吉。病禱安。求財少。問名難。

婚可合。行人遣。宜向善。保團欒。


聖意二

訟和吉。病禱安。名雋望。財頗難。婚可合。行人遣。宜向善。保團圓。


東坡解

作善降祥。作惡降殃。何必禱神。當自揣量。

公心莫昧。勉為善良。前程遠大。可保安康。


碧仙註一

爾欲陷人而自陷。欺心暗裡鬼神知。

若能改善求天福。轉禍為祥事事宜。


碧仙註二

計欲陷人終自陷。欺心暗裡鬼神知。

若能改善求天福。轉禍為祥事事宜。


解曰

此籤顯然說。主人家人有害人之心。而又受人之害。

若能悔過則吉。訟宜和。病祈福。求財少。名利難。婚可合。

只要心正。無事不吉也。


釋義

人之為善為惡。人雖不知。己所獨知。若使欺其不知。而欲害之。

不惟不能害人。抑反為人防害。不可不戒。

占者當公平正大。不可暗昧欺人。則凡事亨通。而無阻滯。

大抵此籤戒人為惡。勉人為善也。


占驗

一人公門不利。占此。乃棄夫為商。大獲厚利。應在三四句。

又晉江鄭生。將赴院試。占此後。果入泮。末句驗甚。



故事

(一)邵堯夫祝香

宋。邵雍。字堯夫。年三十。從父徒居河南。

於書無所不讀。精易理天文術數。居慎。

凡所祝香。皆為國為親之事。未嘗私禱及己也。

初除試。將作監主簿不起。繼除穎州團練推官不就。

卒贈秘書省。著作郎元祐問。諡康節。


(二)周孝侯射虎斬蛟

周處自幼臂力過人，好生事端，四鄰叫苦不疊

，父老將他與南山猛虎，常橋惡蛟並稱“三害”。稍長，翻然悔悟，立志除害補過，並拜師讀書。後任官為朝庭效力。元康元年（296）十一月率兵出征平西。終因梁王用心險惡，援兵不至，弦盡矢絕，力戰而死。建武元年（304）冬，追認平西將軍、清流亭侯，

孝，後人稱“周孝侯”。
